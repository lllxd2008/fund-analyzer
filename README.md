# 基金持有分析助手 - Windows 桌面应用

一个强大的基金投资组合分析工具，支持按基金类型和投资主题进行统计分析。

## 功能特性

- 📁 **上传 Excel 文件** - 支持您的固定格式基金持有清单
- 🏷️ **智能分类** - 自动识别基金类型和投资主题
- 📊 **数据可视化** - 支持饼图和柱状图展示
- 📈 **详细统计** - 按基金类型和投资主题统计分析
- 💾 **导出报告** - 支持导出为 Excel 和 PDF 格式
- 🎨 **现代化界面** - 专业的 Windows 桌面应用设计

## 安装

### 方式 1：使用预编译的 .exe 文件（推荐）

从 [Releases](https://github.com/yourusername/fund-analyzer/releases) 页面下载最新的 `基金持有分析助手-Setup.exe`，然后运行安装程序。

### 方式 2：从源代码构建

#### 前置条件

- Node.js 16.0 或更高版本
- npm 或 yarn

#### 构建步骤

1. 克隆项目

```bash
git clone https://github.com/yourusername/fund-analyzer.git
cd fund-analyzer-electron
```

2. 安装依赖

```bash
npm install
```

3. 运行开发版本

```bash
npm start
```

4. 构建 Windows 应用

```bash
npm run build:win
```

构建完成后，您将在 `dist/` 目录中找到：
- `基金持有分析助手-Setup.exe` - 安装程序
- `基金持有分析助手.exe` - 便携式可执行文件

## 使用方法

1. **打开应用** - 启动基金持有分析助手
2. **上传文件** - 点击"选择 Excel 文件"或拖放您的基金持有清单
3. **查看分析** - 应用自动分析并显示：
   - 总资产和基金数量
   - 按基金类型分布
   - 按投资主题分布
   - 详细的基金信息表
4. **导出报告** - 点击"导出 Excel"或"导出 PDF"保存分析结果

## Excel 文件格式

应用支持以下列名的 Excel 文件：

| 列名 | 说明 |
|------|------|
| 序号 | 基金序号 |
| 基金代码 | 基金代码（6位数字） |
| 基金名称 | 基金名称 |
| 资产情况（结算币种） | 持有金额（单位：元） |
